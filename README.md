# horror-watcher-app

App dedicated only to horror movies.
It has a login page, registration, movie browser and personal section to view the movies you have saved in favorites or pending to see.

## How to start app:

`npm install` or `yarn`

And then: 

`npm run dev` 

# Built with

<li>Vite</li>
<li>Typescript</li>
<li>React</li>
<li>React-Router</li>
<li>React-redux</li>
<li>Redux-toolkit</li>
<li>CSS</li>
<li>Firebase</li>
